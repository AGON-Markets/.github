# AGON

AGON is the Universal Prediction Market Execution Layer — a multi-chain, capital-light, AI-agent-first infrastructure stack that sits above every prediction market venue, protocol, and chain simultaneously. It is not a venue competing for traders. It is the layer that unifies all venues for users, builders, and autonomous agents.

The architecture is a three-layer stack. Layer 1 is the consumer aggregator and universal routing layer: AGON reads odds from Polymarket, Kalshi, Limitless, Azuro, and Drift in real time, then executes cross-chain trades on behalf of users via `AgonRouter.sol`, charging a blended 0.20% routing fee and minting composable ERC-20 wrapped positions (`agonYES-*` tokens) portable across Aerodrome, Morpho, and Yearn. Layer 2 is the B2B infrastructure stack: AI Oracle as a Service for sportsbooks and PM protocols ($99-699/mo), AGON Markets API, a Launchpad for new PM venues, and the MCP Network for AI agents (per-call USDC billing pre-TGE). Layer 3 is AGON's own specialist vertical markets — sport in-game markets (World Cup 2026), crypto-native, FR presidential 2027 — deployed on Base using CPMM AMM, ConditionalTokens ERC-1155, and AI Oracle resolution under 60 seconds.

The capital-light principle is absolute: AGON holds zero treasury cross-chain. All cross-chain liquidity is sourced from existing PM providers. Gross margin is 98%+ vs 30-40% for market-making operators. The AI-agent-first thesis is a structural first-mover advantage: no other PM venue or aggregator exposes an MCP server (Anthropic standard), enabling autonomous agents to discover odds, quote liquidity, and execute trades with on-chain settlement. Multi-chain is live at mainnet day 1: Base + Polygon operational from Week 12, Solana at Month 12.
