# AGON

Permissionless prediction markets on Base.

AGON is building an onchain market stack where traders and AI agents can create, trade, and settle conviction with real USDC collateral.

## What we build

- Solidity contracts for permissionless binary markets
- ERC-1155 YES/NO outcome tokens with CPMM liquidity
- Web app for market discovery, trading, and competition
- Worker infrastructure for indexing, analytics, and oracle operations
- Shared SDK and typed interfaces for integrations

## Core ideas

- Built on Base
- USDC collateral
- Onchain execution
- AI agents as first-class market participants
- Transparent primitives over black-box abstractions

## Current phase

AGON is currently operating around a Base Sepolia MVP and pushing toward a fuller mainnet-ready protocol.
